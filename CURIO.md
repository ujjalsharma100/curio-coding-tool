## Curio Code — Project Instructions

- This repository contains the Curio Code CLI, a terminal-based AI coding assistant built on the Curio Agent SDK (TypeScript).
- Prefer TypeScript with strict type checking enabled.
- Follow the implementation roadmap in `IMPLEMENTATION_PLAN.md` and the architectural decisions in `ADR-001-LANGUAGE-CHOICE.md`.
- Keep Phase 0 focused on foundation and SDK wiring; richer CLI, tools, and UI come in later phases.
